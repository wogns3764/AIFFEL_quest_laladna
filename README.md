## AIFFEL_quest_rs

- 모두의 연구소 AIFFEL 16기 과정 중 진행하는 프로젝트와 피어리뷰를 관리하는 Repo 입니다.

- 기본적인 틀을 제공하며, 파일이나 폴더를 수정 및 업로드하여 사용합니다.


주의사항
1. 각 Quest 폴더 내 Jupyter Notebook을 수정/추가할 때는 커밋 메시지를 상세히 작성해주세요.
2. 불필요한 임시 파일이나 체크포인트 파일(`.ipynb_checkpoints`)는 `.gitignore`에 포함시키는 것을 권장합니다.
